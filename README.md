# LET'S VISION 2026 展商资料库

这是一个面向团队协作的展商资料整理项目，用于沉淀 `LET'S VISION 2026` 相关展商信息、联系人、业务方向和后续跟进记录。

项目采用以下协作模式：

- `Markdown` 作为资料源
- `Python` 脚本自动生成交互式 HTML 导航页
- `Git` 作为版本管理工具
- 后续可发布为统一网页供团队浏览

## 核心约定

### 1. `介绍卡.md` 是唯一资料源

每个展商文件夹里的 `介绍卡.md` 是该展商的基础资料文件。  
团队成员应优先修改这里，而不是直接修改 `展商导航.html`。

### 2. `展商导航.html` 不手工编辑

`展商导航.html` 是自动生成的展示页，任何结构和内容调整都应通过：

- 修改各展商的 `介绍卡.md`
- 或修改 `生成展商导航.py`

然后重新生成 HTML。

### 3. 一次协作建议只改一类内容

例如：

- 只补联系人
- 只补公司业务介绍
- 只补公开链接
- 只补跟进记录

这样更利于合并和回溯。

## 本地使用方法

### 更新展商介绍卡后，重新生成导航页

```bash
python3 生成展商导航.py
```

生成完成后，会更新：

- `展商导航.html`

### 打开展示页

直接在本地打开：

- `展商导航.html`

它支持：

- 搜索展商名称、关键词、联系人
- 按分区筛选
- 按联系人状态筛选
- 点击卡片查看详情
- 直接跳转到对应 `介绍卡.md`

## 推荐协作流程

1. 拉取最新代码
2. 修改对应展商的 `介绍卡.md`
3. 运行 `python3 生成展商导航.py`
4. 检查 `展商导航.html` 是否正常
5. 提交修改并推送
